原生应用（Native App）

使用 android 或者 ios 等原生语言开发的应用，在开发的时候使用开发框架自带的组件进行开发。

  1. android 使用 java
  2. ios 使用 oc
    做原生应用开发的时候有一个岗位叫 app 开发工程师（android开发工程师和ios开发工程师）

还有一种应用叫 windows phone

Hybird 混合应用（轻应用或者web app）

使用网页的形式开发成移动端的手机网站（页面高度相似与手机应用，）通过工具打包成手机app

常用的有：HBuilder，使用原生应用开发工具，Covdva。可以通过注册事件的形式实现原生与网页进行交互。

可以通过接口的方式实现原生应用和网页之间的通信。

原生应用提供的可以调用的方法都有原生应用开发人员提供。和前端工作人员没关系。

原生通过 webview显示页面
